# KayRepo - Dalamud Plugin Repository

This is the official repository for Kay's Dalamud plugins.

## How to use / Comment utiliser
Add the following URL to your Dalamud "Custom Plugin Repositories":
`https://raw.githubusercontent.com/KayrozDbs/KayRepo/main/pluginmaster.json`

## Plugins included / Plugins inclus :
- **Kay Dps**: Real-time performance tracking overlay / Overlay de performance en temps réel.
- **KayArsenal**: Quick desynthesis for unequipped items / Recyclage rapide des équipements.
